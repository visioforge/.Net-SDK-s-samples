# Video Edit SDK X .Net - output-gif (C#/WPF)

Este fragmento demuestra cómo renderizar un archivo de video a un GIF animado utilizando `VideoEditCoreX` y `GIFOutput`. La salida GIF no tiene contenedor — `gifenc` escribe directamente el archivo `.gif` final. Las pistas de audio no se codifican en el GIF.

## Frameworks soportados

* .Net 4.7.2
* .Net Core 3.1
* .Net 5
* .Net 6
* .Net 7
* .Net 8
* .Net 9
