# Video Edit SDK X .Net - output-gif (C#/WPF)

This snippet demonstrates how to render a video file to an animated GIF using `VideoEditCoreX` and `GIFOutput`. GIF output has no container — `gifenc` writes the final `.gif` file directly. Audio tracks are not encoded into the GIF.

## Supported frameworks

* .Net 4.7.2
* .Net Core 3.1
* .Net 5
* .Net 6
* .Net 7
* .Net 8
* .Net 9
