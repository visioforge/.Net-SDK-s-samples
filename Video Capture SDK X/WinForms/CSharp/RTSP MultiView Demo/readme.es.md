# Video Capture SDK X .Net - RTSP MultiView Demo (C#/WinForms)

Esta aplicación soporta descubrimiento y control de cámaras ONVIF, soporta transmisión de ultra baja latencia.

## Frameworks soportados

* .Net 4.7.2
* .Net Core 3.1
* .Net 5
* .Net 6
* .Net 7
* .Net 8
* .Net 9
