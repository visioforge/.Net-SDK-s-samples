# Video Capture SDK X .Net - Networks Streamer (C#/WPF)

This application demonstrates the SDK capabilities.

## Used media blocks

* `H264EncoderBlock` - H.264/AVC video encoding
* `AACEncoderBlock` - AAC audio encoding

## Supported frameworks

* .Net 4.7.2
* .Net Core 3.1
* .Net 5
* .Net 6
* .Net 7
* .Net 8
* .Net 9
