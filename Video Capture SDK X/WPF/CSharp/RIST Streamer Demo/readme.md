# RIST Streamer Demo (Video Capture SDK X, WPF)

A WPF demo application that captures video and audio from local devices and streams them over the RIST (Reliable Internet Stream Transport) protocol using MPEG-TS containers.

## Features

- Camera video capture with format and frame rate selection
- Audio capture device selection with format configuration
- RIST streaming with configurable address and port
- Real-time video preview during streaming
- H264 video and AAC audio encoding

## Required NuGet Packages

- VisioForge.DotNet.Core.MVSX
- VisioForge.CrossPlatform.Core.Windows.x64
- VisioForge.CrossPlatform.Libav.Windows.x64

## Usage

1. Select a video source and format
2. Select an audio source and format
3. Enter the destination address and port
4. Click Start to begin streaming
5. Click Stop to end the stream
