# Video Capture SDK X .Net - Video Compositor Demo (C#/WPF)

This application applies chroma key (green screen) compositing.

## Used media blocks

* `H264EncoderBlock` - H.264/AVC video encoding

## Supported frameworks

* .Net 4.7.2
* .Net Core 3.1
* .Net 5
* .Net 6
* .Net 7
* .Net 8
* .Net 9
