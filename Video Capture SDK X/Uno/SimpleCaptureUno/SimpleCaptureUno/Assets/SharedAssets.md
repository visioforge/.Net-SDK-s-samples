# Shared Assets

These assets must be updated all at once using the [Uno.Resizetizer](https://platform.uno/docs/articles/external/uno.resizetizer/doc/using-uno-resizetizer.html) tool.

This tool will automatically resize images to fit various screen sizes and pixel densities.

## Icon

The app icon (icon.svg) is used for the application icon on all platforms.

## Splash Screen

The splash screen (splash_screen.svg) is used during app startup.
