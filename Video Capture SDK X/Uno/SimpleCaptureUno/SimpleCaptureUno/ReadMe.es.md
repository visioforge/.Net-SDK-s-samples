# Simple Capture Uno - Video Capture SDK X

Este ejemplo demuestra cómo utilizar VisioForge Video Capture SDK X con Uno Platform para aplicaciones de captura de video multiplataforma.

## Características

- Vista previa de cámara
- Captura de Audio/Video a archivo MP4
- Selección de dispositivos de cámara, micrófono y altavoces
- Soporte multiplataforma (Windows, Android, iOS, macOS)

## Página del Producto

[VisioForge Video Capture SDK .NET](https://www.visioforge.com/video-capture-sdk-net)

## Requisitos

- .NET 10.0
- Uno Platform 6.4+
- Licencia VisioForge SDK para uso en producción
