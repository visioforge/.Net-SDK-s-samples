# Video Capture SDK X .Net - video-view-rotate-wpf (C#/WPF)

Esta aplicación demuestra la funcionalidad de rotación de video usando Windows Presentation Foundation (WPF).

## Características

* Vista previa de webcam en vivo con rotación en tiempo real
* Control del ángulo de rotación mediante deslizador (0-360 grados)
* Opción de estiramiento para rotaciones de 90/270 grados
* Preajustes de rotación comunes (0°, 90°, 180°, 270°)

## Frameworks soportados

* .Net 6 (Windows)
* .Net 7 (Windows)
* .Net 8 (Windows)
* .Net 9 (Windows)
* .Net 10 (Windows)
