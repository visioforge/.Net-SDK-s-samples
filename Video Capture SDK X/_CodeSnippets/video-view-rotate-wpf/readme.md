# Video Capture SDK X .Net - video-view-rotate-wpf (C#/WPF)

This application demonstrates video rotation functionality using Windows Presentation Foundation (WPF).

## Features

* Live webcam preview with real-time rotation
* Rotation angle control via slider (0-360 degrees)
* Stretch option for 90/270 degree rotations
* Common rotation presets (0째, 90째, 180째, 270째)

* .Net 6 (Windows)
* .Net 7 (Windows)
* .Net 8 (Windows)
* .Net 9 (Windows)
* .Net 10 (Windows)
