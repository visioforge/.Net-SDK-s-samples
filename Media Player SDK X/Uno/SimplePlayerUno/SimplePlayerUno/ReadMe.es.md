# Simple Player Uno - Media Player SDK X

Este ejemplo demuestra cómo utilizar VisioForge Media Player SDK X con Uno Platform para aplicaciones de reproducción de video multiplataforma.

## Características

- Reproducción de video
- Controles de Reproducir, Pausar, Detener
- Capacidad de búsqueda (seeking)
- Soporte multiplataforma (Windows, Android, iOS, macOS)

## Página del Producto

[VisioForge Media Player SDK .NET](https://www.visioforge.com/media-player-sdk-net)

## Requisitos

- .NET 10.0
- Uno Platform 6.4+
- Licencia VisioForge SDK para uso en producción
