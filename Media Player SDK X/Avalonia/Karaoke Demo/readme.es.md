# Media Player SDK X .Net - Karaoke Demo (C#/Avalonia)

Esta aplicación multiplataforma demuestra la reproducción de karaoke usando archivos CDG (CD+Graphics). Soporta la carga desde un archivo ZIP o archivos CDG y audio separados, con ajuste de tono en tiempo real y control de volumen.

## Frameworks soportados

* .Net 10 (Windows, macOS, Linux)
