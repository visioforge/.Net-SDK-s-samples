# Media Player SDK X .Net - Karaoke Demo (C#/Avalonia)

This cross-platform application demonstrates karaoke playback using CDG (CD+Graphics) files. It supports loading from a single ZIP archive or separate CDG and audio files, with real-time pitch shifting and volume control.

## Supported frameworks

* .Net 10 (Windows, macOS, Linux)
