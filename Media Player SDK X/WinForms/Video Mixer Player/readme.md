# Media Player SDK X .Net - Video Mixer Player (C#/WinForms)

This application plays media files using the universal source decoder.

## Used media blocks

* `UniversalSourceBlock` - Universal media file playback

## Supported frameworks

* .Net 4.7.2
* .Net Core 3.1
* .Net 5
* .Net 6
* .Net 7
* .Net 8
* .Net 9
