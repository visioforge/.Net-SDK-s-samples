# Media Blocks SDK .Net - DataProcessor (C#/Console)

Esta aplicación genera fotogramas de video sintéticos para pruebas y benchmarking, guarda la salida en formato MP4.

## Bloques de medios utilizados

* `VirtualVideoSourceBlock` - Synthetic video generation
* `H264EncoderBlock` - H.264/AVC video encoding
* `MP4SinkBlock` - MP4 file output

## Frameworks soportados

* .Net 4.7.2
* .Net Core 3.1
* .Net 5
* .Net 6
* .Net 7
* .Net 8
* .Net 9
