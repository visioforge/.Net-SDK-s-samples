# Simple Player Uno - Media Blocks SDK

Este ejemplo demuestra cómo utilizar VisioForge Media Blocks SDK con Uno Platform para la reproducción de video.

## Características

- Reproducción de video utilizando tubería de Media Blocks
- Controles de Reproducir, Pausar, Detener
- Capacidad de búsqueda (seeking)
- Soporte multiplataforma (Windows, Android, iOS, macOS)

## Página del Producto

[VisioForge Media Blocks SDK .NET](https://www.visioforge.com/media-blocks-sdk-net)

## Requisitos

- .NET 10.0
- Uno Platform 6.4+
- Licencia VisioForge SDK para uso en producción
