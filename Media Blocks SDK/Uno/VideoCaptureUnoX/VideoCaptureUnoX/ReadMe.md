# Video Capture Uno X - Media Blocks SDK

This sample demonstrates how to use the VisioForge Media Blocks SDK with Uno Platform for building flexible video capture pipelines.

## Features

- Camera preview using Media Blocks pipeline
- Audio/Video capture to MP4 file
- Device selection (Camera, Microphone, Speakers)
- Cross-platform support (Windows, Android, iOS, macOS)

## Product Page

[VisioForge Media Blocks SDK .NET](https://www.visioforge.com/media-blocks-sdk-net)

## Requirements

- .NET 10.0
- Uno Platform 6.4+
- VisioForge SDK license for production use
