# Video Capture Uno X - Media Blocks SDK

Este ejemplo demuestra cómo utilizar VisioForge Media Blocks SDK con Uno Platform para construir tuberías (pipelines) de captura de video flexibles.

## Características

- Vista previa de cámara utilizando tubería de Media Blocks
- Captura de Audio/Video a archivo MP4
- Selección de dispositivos (Cámara, Micrófono, Altavoces)
- Soporte multiplataforma (Windows, Android, iOS, macOS)

## Página del Producto

[VisioForge Media Blocks SDK .NET](https://www.visioforge.com/media-blocks-sdk-net)

## Requisitos

- .NET 10.0
- Uno Platform 6.4+
- Licencia VisioForge SDK para uso en producción
