# RTSP Viewer Uno - Media Blocks SDK

This sample demonstrates how to use the VisioForge Media Blocks SDK with Uno Platform to create an RTSP stream viewer.

## Features

- RTSP stream playback
- ONVIF support
- Low latency mode (60-120ms)
- Username/Password authentication
- Cross-platform support (Windows, Android, iOS, macOS)

## Product Page

[VisioForge Media Blocks SDK .NET](https://www.visioforge.com/media-blocks-sdk-net)

## Requirements

- .NET 10.0
- Uno Platform 6.4+
- VisioForge SDK license for production use
