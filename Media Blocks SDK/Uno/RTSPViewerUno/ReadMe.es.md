# RTSP Viewer Uno - Media Blocks SDK

Este ejemplo demuestra cómo utilizar VisioForge Media Blocks SDK con Uno Platform para crear un visor de transmisiones RTSP.

## Características

- Reproducción de transmisiones RTSP
- Soporte ONVIF
- Modo de baja latencia (60-120ms)
- Autenticación con Usuario/Contraseña
- Soporte multiplataforma (Windows, Android, iOS, macOS)

## Página del Producto

[VisioForge Media Blocks SDK .NET](https://www.visioforge.com/media-blocks-sdk-net)

## Requisitos

- .NET 10.0
- Uno Platform 6.4+
- Licencia VisioForge SDK para uso en producción
