# Media Blocks SDK .Net - Video Mixer Demo (C#/WinForms)

This application plays media files using the universal source decoder, composites multiple video sources into a single output.

## Used media blocks

* `UniversalSourceBlock` - Universal media file playback
* `VideoCompositorBlock` - Multi-source compositing
* `VideoRendererBlock` - Real-time video display

## Supported frameworks

* .Net 4.7.2
* .Net Core 3.1
* .Net 5
* .Net 6
* .Net 7
* .Net 8
* .Net 9
