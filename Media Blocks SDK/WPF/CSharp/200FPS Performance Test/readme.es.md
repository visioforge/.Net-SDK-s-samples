# Media Blocks SDK .Net - 200FPS Performance Test (C#/WPF)

Esta aplicación genera fotogramas de video sintéticos para pruebas y benchmarking, captura fotogramas individuales para análisis o procesamiento, almacena fotogramas entre etapas del pipeline, demuestra procesamiento de video de alto rendimiento a 200 FPS.

## Bloques de medios utilizados

* `VirtualVideoSourceBlock` - Synthetic video generation
* `VideoSampleGrabberBlock` - Frame capture/analysis
* `QueueBlock` - Frame buffering
* `VideoRendererBlock` - Real-time video display

## Frameworks soportados

* .Net 4.7.2
* .Net Core 3.1
* .Net 5
* .Net 6
* .Net 7
* .Net 8
* .Net 9
