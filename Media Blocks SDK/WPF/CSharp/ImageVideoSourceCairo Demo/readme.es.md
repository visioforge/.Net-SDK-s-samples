# Media Blocks SDK .Net - ImageVideoSourceCairo Demo (C#/WPF)

Esta aplicación demuestra las capacidades del SDK.

## Bloques de medios utilizados

* `ImageVideoSourceCairoBlock` - Cairo-based image video source
* `VideoRendererBlock` - Real-time video display

## Pipeline

```mermaid
graph LR
    ImageVideoSourceCairoBlock -- video --> VideoRendererBlock
```
