# Media Blocks SDK .Net - Image Sequence Source Demo (C#/WPF)

Esta aplicación crea video desde secuencias de imágenes.

## Bloques de medios utilizados

* `ImageSequenceSourceBlock` - Image sequence playback
* `VideoRendererBlock` - Real-time video display
* `AudioRendererBlock` - Real-time audio playback

## Pipeline

```mermaid
graph LR
    ImageSequenceSourceBlock -- video --> VideoRendererBlock
```
